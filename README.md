# EverJar

🫙 **One note a day. A lifetime of love.**

EverJar is a shared digital memory jar where people capture love, thoughts, and moments—one note at a time—building a meaningful collection over days, months, and years.

## Features

- 🫙 Shared "Jar" with a partner
- 💌 One-note-per-day ritual
- 🔒 Locked future notes ("open on this date")
- 📊 365-day journey tracker
- 🎁 Export jar as a gift

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run web
   ```

3. Open [http://localhost:8081](http://localhost:8081) in your browser.

## Monetization

- **Free**: Basic jar + daily notes
- **Premium**: Unlimited jars, media (photos/audio), custom themes, printable "memory book"

## Target Users

- Couples 💑
- Long-distance relationships 🌍
- Close friends 🫂
- Self-journaling users
